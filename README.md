This is a website I made using Firebase and the TMDB API. The premise is that once you make an account, you can send other users that have accounts friend requests. Once they accept you will become friends and can send/receive recommendations about what movies to watch. This is done by searching for a movie in the searchbar/clicking on one of the top 10 trending movies showing in the homepage, selecting which friend you want to recommend the movie to and pressing Send. You can see what recommendations you have received by clicking on the little star located in the navbar.
